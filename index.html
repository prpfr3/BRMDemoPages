Draft example pages for the Bluebell Railway Museum Website and/or Interactive Display

Note that this is not the live Museum Website which can be found here.

Bluebell Railway Timeline

For the Website and/or Interactive Display Developed using the same standard web page software as the museum website (HTML,CSS,JS) + US North Western University's Knightlab open source Timeline package. Very easy to adapt the content. Look and feel can be adapted but within limits. Would work on an interactive screen:- Swipe left and right on the pages and/or manipulate the timeline in the footer. Content of this example requires further work and rationalisation. Very easy to apply to other content, for example, a timeline with a page for each of the LBSCR CMEs, a timeline for each of the Bluebell railway locomotives, a timeline for key museum exhibits in chronological order and so on.

Bluebell Railway Storymap

For the Website and/or Interactive Display Visitors can step along the Bluebell line to find out about the stations etc on the route. Similar to the timeline but uses the Storymap package rather than the Timeline package. Content shown is currently from Wikipedia. Basic content available to generate such storymaps for any route in the UK (e.g. could have storymaps for other LBSCR routes, named train routes such as the Atlantic Coast Express etc). Another example is here

Museum Map

For Interactive Display View a museum plan, key exhibits and get more info about the key exhibits. This example is a very rough draft, possibly with some software bugs. As no museum plan readily available, a plan of the Sheffield Park area has been used. This approach could also be used to display a map/plan from the archive and describe points of detail.
